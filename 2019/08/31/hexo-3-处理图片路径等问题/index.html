hexo-3-解决图片路径方法

发表于 2019-08-31

修改配置与安装插件

1.首先站点配置文件里的 post_asset_folder:设置为true

2.在项目根目录下 package.json 中添加配置

3.项目根目录下，安装插件

4.使用

新建hexo new "新文章"

之后运行hexo n “博客名”来生成md博客时，会在_post目录下看到一个与博客同名的文件夹
将想要上传的图片先扔到文件夹下，然后在博客中使用markdown的格式引入图片 格式：![你想要输入的替代文字](xxxx/图片名)
