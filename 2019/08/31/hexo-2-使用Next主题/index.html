hexo-2-使用与配置Next主题

  1. 下载 next 主题 git clone https://github.com/theme-next/hexo-theme-next themes/next
  2. 修改站点 _config.yml 配置 theme: next
  3. 修改主题 _config.yml 配置 scheme: Gemini
    ok~ hexo clean && hexo s

下载next主题并使用

1.下载主题

(文件根目录下bit bash 下输入后安心等待)
不要用 https://github.com/iissnan/hexo-theme-next 这个链接,因为这个是2年前的next了, 至于后面的themes/next就是在根目录的themes下建next文件夹的命令行表达形式,当然若是真下了老版的next, cd themes/next 后git pull更新至最新版本即可
